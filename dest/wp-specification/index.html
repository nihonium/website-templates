カスタム投稿・カスタムタクソノミー(カスタム分類)の設定

【使用プラグイン】：Custom Post Type UI
【実装した主な内容】：
・4つの投稿タイプを登録しています(お知らせ、製品情報、サービス、採用情報)。
・「お知らせ」には3つのカテゴリを設定しています(その他、トピックス、プレスリリース)。
・「採用情報」には4つのカテゴリを設定しています(中途採用、契約社員採用、新卒採用、障がい者採用)。
・「archive-***.php」「taxonomy-***.php」側で、カスタム投稿やカスタムタクソノミーごとに一覧で出力されるよう実装しています。
【参考URL】https://webst8.com/blog/wordpress-custom-post-type-ui/

カスタムフィールドの設定

【使用プラグイン】：Advanced Custom Fields
【実装した主な内容】：
・「TOPページ___MVスライダー」「カスタム投稿ページ___キービジュアル」「カスタム投稿ページ___リード」の3つのフィールドグループを実装しています。
・画像は「functions.php」で指定したサイズから、「single.php」側でリサイズして出力されます。
【参考URL】http://galileo-venus.com/2020/05/14/20200507_wp_acf01/

お問い合わせフォーム設定

【使用プラグイン】：Contact Form 7、Contact Form 7 Multi-Step Forms
【実装した主な内容】：
・「Contact Form 7」の拡張プラグインである「Contact Form 7 Multi-Step Forms」を使って、お問い合わせの確認ページと完了ページを実装しています。
・「function.php」側で、reCAPTCHA v3をお問い合わせページのみで表示される様に設定しています。
【参考URL】https://wpmake.jp/contents/plugin/contact-form7/add-confirm/

セキュリティ対策について

【使用プラグイン】：SiteGuard WP Plugin
【実装した主な内容】：
・ログインページに画像認証を追加しています。
・WordPressの管理画面(ダッシュボード)に入る度メールで通知でされ、SiteGuard WP Pluginのダッシュボードからログイン履歴が参照できます。
・「function.php」側で、クリックジャキング対策を行っています(X-FRAME-OPTIONS)。
【参考URL】https://www.jp-secure.com/siteguard_wp_plugin/
